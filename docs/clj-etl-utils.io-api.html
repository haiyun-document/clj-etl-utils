byte-marker-matches?

  function

  
Usage: (byte-marker-matches? marker-bytes file-bytes)


  
Test if a given marker is equivalent to the given set of
bytes from the file.  This is a prefix test, and will return true of
the shorter marker matches the beginning of the longer marker.
